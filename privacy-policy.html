PesquisaPlanilha

Extensao para Google Chrome - Politica de Privacidade

Ultima atualizacao: marco de 2026
Resumo: A extensao PesquisaPlanilha nao coleta, armazena nem transmite nenhum dado pessoal para servidores externos. Todo o processamento ocorre localmente no seu navegador.

1. O que a extensao faz

O PesquisaPlanilha e uma extensao para Google Chrome que permite pesquisar termos em planilhas Excel abertas no Microsoft OneDrive ou SharePoint Online, diretamente pelo navegador. Retorna resultados com numero do processo, data e assunto.

2. Dados acessados

Para funcionar, a extensao acessa:

  • Conteudo das celulas da planilha - lido via DOM da aba do Excel Online aberta pelo usuario, exclusivamente na sessao ativa.
  • URLs das abas abertas no Chrome - usadas apenas para identificar quais abas contem uma planilha do OneDrive ou SharePoint.

Nenhum outro dado e acessado.

3. Armazenamento local

Os dados lidos da planilha sao armazenados temporariamente no armazenamento local do Chrome (chrome.storage.local), exclusivamente no dispositivo do usuario. Esses dados nunca saem do dispositivo e podem ser apagados a qualquer momento desinstalando a extensao.

4. O que NAO fazemos

  • Nao transmitimos nenhum dado para servidores externos.
  • Nao coletamos dados pessoais, de identificacao ou de uso.
  • Nao usamos cookies, rastreadores ou ferramentas de analytics.
  • Nao compartilhamos dados com terceiros.
  • Nao exibimos anuncios.
  • Nao requeremos login ou conta de qualquer servico.

5. Permissoes utilizadas

  • tabs - identificar abas abertas com Excel Online/OneDrive.
  • scripting - injetar script de leitura na aba selecionada para extrair o conteudo das celulas.
  • storage - salvar os dados da planilha no armazenamento local do Chrome entre sessoes.
  • host permissions (*.sharepoint.com, onedrive.live.com, *.office.com) - necessarias para executar o script nas paginas do Excel Online.

6. Seguranca

Todo o processamento ocorre localmente no dispositivo do usuario. A extensao nao possui servidor proprio e nao estabelece conexoes de rede alem das realizadas pelo proprio Chrome ao carregar as paginas do OneDrive.

7. Alteracoes nesta politica

Caso esta politica seja atualizada, a nova versao sera publicada nesta pagina com a data de atualizacao revisada.

8. Contato

Duvidas podem ser enviadas atraves da pagina da extensao na Chrome Web Store, usando o link de suporte disponivel na listagem.
